# 6. Функции

### 6.1. Функции - объекты высшего порядка

```
function myFunction(a, b) {
    return a + b;
}

myFunction(1, 2);
```

### 6.2. Анонимные функции

```
let fn = function (a, b) {
    return a + b;
}

fn(1, 2);
```

### 6.3. Стрелочные функции

```
let fn = (a, b) => {
    return a * b;
}

fn(2, 3);
```

### 6.4. callback

```
let array = [1, 2, 3];

array.forEach((item, i) => {
    console.log(item, i);
});

```
