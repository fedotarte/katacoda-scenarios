### 6.1. Функции - объекты высшего порядка

```

}

```

### 6.7. Асинхронные генераторы

```
async function *generator() {
    yield 1;
    yield 2;
    yield 3;
} 

(async () => {

    for await (let item of generator())
        console.log(item);

})();


```
